# 🎮 Guia de Uso

## Iniciando o Programa

```bash
python3 esc.py
```

## Menu Principal

Você verá um banner bonito e receberá 3 opções:

### 1️⃣ Digitar o Texto Agora

Escolha esta opção para digitar seu texto diretamente no programa.

```
Digite seu texto (pressione Enter 2x para terminar):
```

- Escreva seu texto normalmente
- Pressione Enter duas vezes para terminar

### 2️⃣ Ler de um Arquivo

Use arquivos `.txt` para grandes textos.

```
📁 Digite o caminho do arquivo (.txt): exemplos/exemplo1.txt
```

- Caminho relativo: `exemplos/exemplo1.txt`
- Caminho absoluto: `/home/user/Documents/meu_texto.txt`

### 3️⃣ Usar Exemplo Padrão

Usa uma história pré-carregada sobre Clara e o sítio.

## Escolher Velocidade

```
1 - 🐢 Lento (0.15s por caractere)
2 - 🚶 Normal (0.05s por caractere) ⭐ RECOMENDADO
3 - 🏃 Rápido (0.02s por caractere)
4 - ⚡ Muito Rápido (0.01s por caractere)
```

**Recomendações:**
- Para visualizar: use **Lento**
- Para uso geral: use **Normal**
- Para testes: use **Rápido**
- Para máxima velocidade: use **Muito Rápido**

## Resumo da Digitação

Antes de iniciar, verá:

- 📊 Total de caracteres
- ⏱️ Tempo estimado
- 📝 Primeira linha do texto

## Iniciando a Digitação

```
⚠️ AVISO: Mude o foco para a janela onde deseja digitar!
Iniciando em: 5... 4... 3... 2... 1...
```

1. Pressione ENTER
2. Rapidamente clique na janela onde deseja digitar (Notepad, VS Code, etc)
3. O texto será digitado automaticamente!

## Barra de Progresso

Durante a digitação, você verá:

```
[████████████████░░░░░░░░░░░░] 54.2% (1250/2307) - 12.5s
```

## Cancelar Digitação

Pressione `CTRL+C` a qualquer momento para cancelar.

## Dicas e Truques

- 💡 Teste primeiro com o exemplo padrão
- 💡 Use "Normal" para a melhor experiência
- 💡 Crie seus próprios arquivos `.txt`
- 💡 Experimente com diferentes aplicativos
